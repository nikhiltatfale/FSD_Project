Firebase Setup

Configure your Firebase credentials to get started

Quick Setup Guide:

  1. Go to Firebase Console
  2. Create a new project or select existing one
  3. Click "Add app" → Choose "Web" () icon
  4. Register your app and copy the config values
  5. Go to "Firestore Database" → Create database (test mode)
  6. Paste your credentials below
